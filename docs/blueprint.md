# **App Name**: Bazaar Showcase

## Core Features:

- Product Image Gallery: Display the primary product image with a set of thumbnail images for quick browsing.
- Product Details: Showcase the product title, price, and detailed description.
- Theme Toggle: Toggle the UI between a dark and light mode
- System Color Scheme: Use the system color scheme for the current dark/light mode. Toggle on the display of a button.

## Style Guidelines:

- Primary color: Soft, modern blue (#A0C4FF) to convey trust and serenity.
- Background color: Light gray (#F0F0F0) for light mode, dark gray (#333333) for dark mode.
- Accent color: Slightly desaturated light blue (#BDB2FF), providing gentle contrast
- Body and headline font: 'Inter' (sans-serif) for clean and modern readability.
- Use simple, outlined icons for clarity and a minimalist aesthetic.
- Maintain a clean and spacious layout to focus attention on the product details.
- Implement subtle transition animations for image gallery changes interactions.
